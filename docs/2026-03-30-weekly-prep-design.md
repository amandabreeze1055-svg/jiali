# 本周穿搭模块设计

## 位置
日历和近期事项之间，默认展开。用户无数据时不显示。

## 布局
- 标题行：「本周穿搭」+ 右侧「编辑」按钮
- 7列网格（周一~周日），每列：星期标签 → 日期 → 分隔线 → 穿搭（如有）→ 携带物品（如有）
- 今天高亮（浅橙背景）

## 数据结构
```js
// 云数据库集合: weekly_prep
{
  _openid: 'xxx',
  weekStart: '2026-03-30',  // 本周一日期
  days: [
    { weekday: 1, outfit: '', items: [] },
    // ... 共7天
  ]
}
```

outfit 和 items 按类型分组显示，空则不显示对应行。

## 编辑模式
- 穿搭：自由文本输入
- 物品：自由文本输入，逗号或换行分隔
- 底部「保存」「取消」按钮，保存写入云数据库

## AI 识别
parseText/parsePhoto 返回 `weekly_prep` 字段时，弹确认窗后写入云数据库。

## 周更新
打开首页检查 weekStart 是否本周一：
- 是 → 直接显示
- 否 → 复制上周数据为模板，更新日期

## 图标映射
制服→👔 运动服→🏃 便装→👕 跆拳道服→🥋 泳衣→🩱 默认→👗

## 存储
云数据库（已启用云开发），支持跨设备同步。
